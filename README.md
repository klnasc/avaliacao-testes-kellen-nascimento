Avaliação de Testes e Versionamento

Descrição:

Projeto desenvolvido para a atividade da Unidade Curricular 03 - Realizar Testes e Manutenção do Sistema de Informação.
O objetivo é aplicar conceitos de versionamento com Git/GitHub e testes unitários utilizando JUnit em Java.

Funcionalidades:

A classe Calculadora possui os seguintes métodos:

soma(int a, int b) - realiza a soma de dois números inteiros.
divisao(int a, int b) - realiza a divisão de dois números inteiros.
Testes Implementados

Os testes unitários validam:

Operações de soma com valores válidos.
Operações de divisão com valores válidos.
Tratamento de exceção para divisão por zero.
Como Executar os Testes

Clone o repositório:
Abra o projeto em uma IDE compatível com Java (IntelliJ IDEA, Eclipse ou VS Code).
Execute a classe CalculadoraTest.java.
Verifique os resultados dos testes no console da IDE.
